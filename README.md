# 🌦️ Weather App

A simple command-line Weather App built with Python that fetches real-time weather information using the OpenWeather API.

**Features**

- Search weather by city name
- Displays:
  -  City
  - Country
  - Temperature (°C)
  - Humidity (%)
  - Weather Description
- Fetches live weather data using an API

**Technologies Used**

- Python 3
- Requests Library
- OpenWeather API
